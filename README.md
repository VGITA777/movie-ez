# Movie Ez

Movie EZ is a web application that allows users to watch movies and TV shows online.
It provides a user-friendly interface to search for and stream content from various sources.

---

## Features

- Search for movies and TV shows
- Watch movies and TV shows online
- Responsive design for desktop, tablet, and mobile devices
- DRM-free (All content is retrieved from public sources)
